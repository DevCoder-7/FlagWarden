# FlagWarden Demo Script

## Goal

Record a polished 30-60 second demo for GitHub, LinkedIn, or freelance bot
developer applications.

## Exact Telegram Commands To Record

1. `/start`
2. Tap `Challenge` or send `/challenge`
3. `/hint`
4. `/answer flag{...}`
5. `/score`
6. `/safety`

## Suggested Timeline

| Time | Action | Caption or Voiceover |
| --- | --- | --- |
| 0-5s | Open the Telegram bot | "FlagWarden helps learners practice CTF safely and ethically." |
| 5-12s | Send `/start` | "The bot introduces its legal lab-only scope and quick actions." |
| 12-24s | Open a challenge | "Learners can request daily or random CTF-style challenges." |
| 24-34s | Request `/hint` | "Hints support learning while reducing the possible score." |
| 34-46s | Submit `/answer flag{...}` | "Answers are validated, explanations are returned, and points are awarded." |
| 46-54s | Send `/score` | "Progress is persisted with score, solved count, and streak tracking." |
| 54-60s | Send `/safety` | "Safety guardrails keep the experience focused on legal CTF labs." |

## Short Caption Script

FlagWarden is a Telegram CTF Cybersecurity Learning Bot for safe, ethical
practice. It supports daily and random challenges, hints, answer validation,
scoring, streak tracking, leaderboards, and safety guardrails for legal
lab-only learning.

## Recording Tools

- Phone screen recorder for native Telegram mobile demos.
- OBS for desktop recordings.
- ShareX for quick Windows screen capture.
- ScreenToGif for lightweight GIF recording on Windows.
- ffmpeg for trimming and converting videos.

## Privacy Checklist

- Use a test Telegram account if possible.
- Crop private usernames, chat lists, and profile photos.
- Hide bot tokens, ngrok URLs, terminal windows, and admin pages.
- Do not show `.env`, BotFather token messages, Meta credentials, or phone numbers.
- Revoke any token accidentally shown in a recording.
