# FlagWarden Screenshot Guide

Use this folder for real Telegram screenshots that show FlagWarden working end
to end.

## Required Files

- `start.png`
- `challenge.png`
- `answer-correct.png`
- `score.png`
- `safety-policy.png`

## Capture Checklist

- Send `/start` and capture the welcome menu.
- Send `/challenge` and capture a challenge prompt.
- Send `/hint` if you want the demo to show hints.
- Submit a correct `/answer ...` and capture the success response.
- Send `/score` and capture progress.
- Send `/safety` and capture the ethical-use policy.

## Privacy Checklist

- Crop private usernames, phone numbers, and chat lists.
- Hide bot tokens, `.env` values, ngrok URLs, Meta credentials, and admin pages.
- Do not show private chats or unrelated contacts.
- If a token appears in a screenshot, revoke it in BotFather before publishing.
- Prefer a clean test Telegram account for portfolio screenshots.
